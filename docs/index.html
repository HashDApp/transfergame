# transfergame # 转账即游戏 这是一款基于EOS转账功能的小游戏 游戏的合约代码完全开源 <font size=6><a href="https://github.com/HashDApp/transfergame" target="_blank">GitHub</a></font> ## 游戏玩法 1、使用任意一款EOS钱包在任意时刻向合约账号 ***transfergame*** 发起任意数额的EOS转账即可参与游戏 2、每11笔转账为一轮游戏，由合约代码在11笔转账中随机选择一名获胜玩家 3、获胜玩家将获得本轮游戏所有的EOS作为奖励 4、玩家可以通过在一轮游戏中发起多次转账，增加随机选择胜出的概率 5、玩家可以通过转账较大数额的EOS，增加随机选择胜出的概率
